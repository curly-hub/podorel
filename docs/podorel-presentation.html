Architecture

Small, understandable pieces with clear boundaries.

A Go web service, a per-user agent, SQLite state, and an Angular UI. Nothing exotic, no cluster requirement, no hidden control plane.

Browser UI Operate pods, templates, Compose stacks, logs, scans, settings, agents, and diagnostics.
Go web/API Handles auth, settings, audit, SQLite state, health checks, and UI delivery.
User agent Runs beside the Linux user and talks to rootless Podman socket or CLI.
SQLite Simple local persistence for agents, sessions, audit logs, scans, templates, and settings.
Podman pod The web service runs in a rootless Podman pod while the agent stays host-side.
Local HTTPS Native TLS and local CA support keep browser security features available.
